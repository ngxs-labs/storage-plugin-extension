In this example we use the InterceptorStrategy class to configure pre-serialization/post-deserialization interceptors within the @ngxs-labs/storage-plugin. The pre-serialization interceptor keeps the width, length and height for a parcel but excludes the description from being serialized. The post-serialization interceptor instantiates a DeliveryStateModel concrete class.
